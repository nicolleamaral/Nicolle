O que são Variáveis CSS?

Sua maior função é armazenar valores que serão utilizados posteriormente no código.

O que é Versionamento de código?

Permite que os arquivos sejam armazenados de forma organizada.

O que é commit?

É uma captura de um instantâneo das mudanças preparadas do projeto no momento.

O que é um deploy em programação?

O Deploy consiste no processo de colocar no ar uma aplicação já concluída.
